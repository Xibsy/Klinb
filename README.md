## Klinb

Это будет смесью блинка и карточек с инфо о человеке. Поговорим о функциях этого сайта: 
1) Лента с загруженными местами, поиск по названию места (можно запросить) 
2) Чат
3) Профили пользователей где они рассказывают о себе и их фоточки Y.Y (это тоже можно запросить)
4) Аватарки (возможно) будут самодельные или нагло спертые из интернета 
А 5, я пока не придумала, наверное на этом все 🤔
